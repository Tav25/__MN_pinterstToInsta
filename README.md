# Pinterest to Instagram

Этот проект представляет собой скрипт для Tampermonkey, который позволяет извлекать и отображать ссылки на `.cmfv` файлы из Pinterest.

## Структура проекта

- `pinterest-to-insta.user.js` - Основной файл скрипта для Tampermonkey. Отвечает за перехват сетевых запросов, отображение UI и привязку ссылок к карточкам пинов.
- `main.js` - Дополнительный файл скрипта с логикой обработки данных.
- `config.json` - Файл конфигурации с настройками для работы скрипта.

## Установка

1. Установите расширение Tampermonkey в вашем браузере.
2. Откройте файл `pinterest-to-insta.user.js` в Tampermonkey.
3. Настройте параметры в файле `config.json` при необходимости.

## Использование

1. Откройте Pinterest в вашем браузере.
2. Скрипт автоматически начнет отслеживать сетевые запросы и отображать найденные `.cmfv` ссылки в плавающей панели.
3. Ссылки также будут привязаны к соответствующим карточкам пинов для быстрого доступа.

## Функции

- **Перехват сетевых запросов**: Скрипт перехватывает запросы `fetch` и `XHR` для поиска URL, содержащих `.cmfv`.
- **Отображение UI**: Создает плавающую панель для отображения найденных `.cmfv`-URL.
- **Привязка к карточкам пинов**: Добавляет чип с ссылкой на `.cmfv` файл к карточке пина.
- **Обработка SPA-навигации**: Обеспечивает работу скрипта при навигации по сайту без перезагрузки страницы.

## UI/UX Цветовая палитра

- Основной цвет: `#0d3b66`
