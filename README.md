# Pinterest to Instagram

Этот проект представляет собой скрипт для Tampermonkey, который позволяет переносить контент из Pinterest в Instagram.

## Структура проекта

- `pinterest-to-insta.user.js` - Основной файл скрипта для Tampermonkey.
- `main.js` - Основной файл скрипта с логикой переноса контента.
- `config.json` - Файл конфигурации с настройками для Pinterest и Instagram.

## Установка

1. Установите расширение Tampermonkey в вашем браузере.
2. Откройте файл `pinterest-to-insta.user.js` в Tampermonkey.
3. Настройте токены доступа в файле `config.json`.

## Использование

1. Откройте Pinterest в вашем браузере.
2. Скрипт автоматически извлечет данные и перенесет их в Instagram.
