# Pinterest to Instagram

Этот проект представляет собой скрипт для Tampermonkey, который позволяет извлекать и отображать **прямые ссылки на видео** из Pinterest и собирать их в удобную панель.

## Структура проекта

- `pinterest-to-insta.user.js` — основной файл скрипта для Tampermonkey. Отвечает за добавление кнопок «Скачать», получение прямых URL на видео и ведение списка ссылок.
- `main.js` — дополнительный файл скрипта с логикой обработки данных.
- `config.json` — файл конфигурации с настройками для работы скрипта.

## Установка

1. Установите расширение Tampermonkey в вашем браузере.
2. Откройте файл `pinterest-to-insta.user.js` в Tampermonkey.
3. Настройте параметры в файле `config.json` при необходимости.

## Использование

1. Откройте Pinterest в вашем браузере.
2. На каждой карточке с видео появится кнопка **«Скачать»**.
3. Справа будет закреплена панель со списком прямых ссылок:
   - Copy all — скопировать все URL.
   - Export .txt — экспорт списка в файл.
   - Remove last — убрать последнюю ссылку.
   - Clear — очистить список.
4. Список сохраняется между перезагрузками страницы.

## Функции

- **Кнопка «Скачать» на видео-пинах**: добавляет overlay-кнопку на карточки с видео.
- **Получение прямых ссылок**: извлекает прямой URL через `PinResource` с фолбеком на страницу пина.
- **Сниффинг сетевых запросов**: перехватывает прямые `.cmfv` URL, если Pinterest не отдает данные через API.
- **Панель-буфер**: хранит очередь ссылок, поддерживает автокопирование, экспорт и очистку.
- **Поддержка бесконечной ленты**: отслеживает новые карточки через `MutationObserver`.
